# เตรียมตัวลงพื้นที่ตรวจสอบ

คู่มือเตรียมตัวก่อนออกไปตรวจสอบสถานีวิทยุ สำหรับทีมตรวจสอบ กสทช. เขต 23

---

## 1. อุปกรณ์ที่ต้องเตรียม ก่อนลงตรวจสถานี FM

### Spectrum Analyzer — R&S FSH8

เครื่อง [Rohde & Schwarz FSH8](https://www.rohde-schwarz.com/us/products/test-and-measurement/handheld/rs-fsh-handheld-spectrum-analyzer_63493-8180.html) เป็น Handheld Spectrum Analyzer สำหรับงานภาคสนาม

**สเปคสำคัญ:**

| รายการ | ค่า |
|--------|-----|
| ย่านความถี่ | 9 kHz - 8 GHz |
| น้ำหนัก | ~3 kg (รวมแบต) |
| แบตเตอรี่ | ใช้งานได้ ~4.5 ชม. |
| หน้าจอ | 6.5 นิ้ว สี TFT |
| ทนทาน | กันน้ำสาด, ทนกระแทก |

**เช็คก่อนออกเดินทาง:**

- [ ] แบตเตอรี่ชาร์จเต็ม (หรือเตรียมแบตสำรอง)
- [ ] สาย AC adapter สำหรับชาร์จในรถ
- [ ] Calibration ยังไม่หมดอายุ
- [ ] ไฟล์ Limit Line / Mask ตั้งค่าไว้แล้ว (SM.1268-2)

### สายอากาศ (Antenna)

**Whip Antenna (สายอากาศแบบแส้)**

สายอากาศแบบแส้ (Whip) สำหรับรับสัญญาณย่าน VHF ทั่วไป ต่อเข้ากับ FSH8 โดยตรงผ่านขั้ว N-Type หรือ SMA

| รายการ | ค่า |
|--------|-----|
| ประเภท | Omnidirectional Whip |
| ย่านใช้งาน | 80 - 200 MHz (ครอบคลุม FM + ย่านการบิน) |
| ขั้วต่อ | SMA หรือ BNC (ใช้ adapter ตามเครื่อง) |
| ใช้กับ | วัด Occupied BW, Deviation, Spurious, สแกนย่าน 87-137 MHz |

**อุปกรณ์เสริม:**

- [ ] สาย SMA-to-BNC adapter (ถ้าต้องใช้)
- [ ] สาย Coaxial สำรอง
- [ ] ขาตั้งสายอากาศ (tripod) สำหรับวัดจุดคงที่

### อุปกรณ์อื่นๆ

| อุปกรณ์ | ใช้ทำอะไร |
|---------|----------|
| กล้องถ่ายรูป / มือถือ | ถ่ายรูปสถานี, เสาอากาศ, ป้าย |
| GPS (หรือมือถือ) | บันทึกพิกัดสถานที่ตรวจ |
| สมุดจด / แบบฟอร์ม | บันทึกผลตรวจเบื้องต้น |
| Notebook / Tablet | ลงข้อมูลในระบบ OPER |
| เสื้อกั๊ก กสทช. | แสดงตัวตนตอนเข้าตรวจสถานี |
| หนังสือมอบหมาย | เอกสารอำนาจในการตรวจ |
| น้ำดื่ม + ของกิน | ลงพื้นที่ทั้งวัน ต้องเตรียมตัว! |

## 2. วางแผนก่อนออกเดินทาง

### ดูแผนงานใน Notion

เปิด [FM Radio Tracker 2569](https://www.notion.so/3305654caeb18106b05ef84111df6843?v=3305654caeb1810fb8db000c71bf5b46) เพื่อดูว่าเดือนนี้ต้องตรวจสถานีไหนบ้าง

**สิ่งที่ต้องเช็ค:**

- [ ] เดือนนี้มี target กี่สถานี
- [ ] สถานีไหนที่ยังไม่ได้ตรวจ (Status: Not started)
- [ ] ใครรับผิดชอบสถานีไหน (Assigned To)
- [ ] จังหวัดไหนบ้าง → วางแผนเส้นทาง

### วางเส้นทาง

!!! tip "เทคนิคจัดเส้นทาง"

    - รวมสถานีที่อยู่ใกล้กัน / จังหวัดเดียวกัน ไว้ในวันเดียว
    - ตรวจสถานีที่ไกลที่สุดก่อนตอนเช้า แล้วค่อยวนกลับ
    - เผื่อเวลาเดินทาง + เวลาตรวจสถานีละ ~30-45 นาที
    - เช็คสภาพอากาศก่อน — ฝนตกหนักอาจวัดค่าไม่แม่น

## 3. ลงพื้นที่ตรวจสถานี

### ใช้ระบบ Make it Fast

เปิด [Make it Fast](https://make-it-fast.vercel.app/) สำหรับบันทึกข้อมูลการตรวจสถานีขณะอยู่ในพื้นที่

### ขั้นตอนตรวจ ณ สถานี

```
1. ถึงสถานี
   └→ ถ่ายรูปสถานี + เสาอากาศ + ป้าย

2. ต่อสายอากาศ + เปิดเครื่อง FSH8
   └→ ตั้งค่าตาม parameter ที่ต้องวัด

3. วัด 3 parameter หลัก
   ├→ Occupied Bandwidth (99% Power method)
   ├→ Frequency Deviation (SM.1268-2 Limit)
   └→ Spurious Emission (สแกน 87-137 MHz)

4. บันทึกผล
   ├→ Save หน้าจอ (screenshot) ทุกการวัด
   ├→ จดค่าที่วัดได้
   └→ ลงระบบ OPER (https://fmr.nbtc.go.th/)

5. สรุป PASS / FAIL
   └→ ถ้า FAIL → ถ่ายรูปหลักฐาน + แจ้งสถานี
```

### Checklist ก่อนออกจากสถานี

- [ ] วัด Occupied BW แล้ว → Save screenshot
- [ ] วัด Frequency Deviation แล้ว → Save screenshot
- [ ] วัด Spurious 87-137 MHz แล้ว → Save screenshot
- [ ] ถ่ายรูปสถานี + เสา + ป้าย แล้ว
- [ ] ลงข้อมูลในระบบ OPER แล้ว
- [ ] อัพเดท Status ใน Notion เป็น "Done" แล้ว

## 4. อุปกรณ์ที่ต้องเตรียม ก่อนลงตรวจสถานี EMF

งาน EMF ต่างจากงาน FM — ไม่ได้ไปตรวจสถานีวิทยุ แต่ไปวัดค่าคลื่นแม่เหล็กไฟฟ้าจากสถานีฐานมือถือ (Cell Site) ในพื้นที่ชุมชน

### ขั้นตอนที่ 1: นัดทีมคุ้มครอง

!!! info "ต้องนัดล่วงหน้า"

    งาน EMF ต้องออกพื้นที่ร่วมกับ **ทีมคุ้มครองผู้บริโภค** เพื่อให้ความรู้ประชาชนในพื้นที่ไปพร้อมกัน

    - นัดวันเวลากับทีมคุ้มครองล่วงหน้า
    - แจ้งพื้นที่ที่จะไป + จำนวนจุดวัด
    - เตรียมเอกสารแจกประชาชน (ถ้ามี)

### ขั้นตอนที่ 2: เตรียมอุปกรณ์ Narda

| อุปกรณ์ | เช็ค |
|---------|-----|
| เครื่อง Narda SRM-3006 / NBM-550 | - [ ] แบตเตอรี่ชาร์จเต็ม |
| โพรบวัด EMF (Isotropic Probe) | - [ ] ต่อกับเครื่องได้ปกติ |
| ขาตั้งสามขา (Tripod) | - [ ] ตั้งได้สูง 1.1 / 1.5 / 1.7 ม. |
| GPS หรือมือถือ | - [ ] บันทึกพิกัด |
| กล้องถ่ายรูป | - [ ] ถ่ายสถานี + เสา + ป้าย |
| เสื้อกั๊ก กสทช. + หนังสือมอบหมาย | - [ ] พร้อม |
| สายชาร์จ / แบตสำรอง | - [ ] เผื่อไว้ |

### ขั้นตอนที่ 3: ดูแผนว่าจะไปที่ไหน

เปิด [EMF & OFF-site Monitoring Planner 2569](https://www.notion.so/b72fa46551f246539cea30f5c24ef68d?v=3315654caeb181ec9b30000c0eee3ac9) ใน Notion

**สิ่งที่ต้องเช็ค:**

- [ ] เดือนนี้ต้องวัดที่ไหน (จังหวัด + พื้นที่)
- [ ] สถานะ: Not started / In progress
- [ ] ใครรับผิดชอบ (ผู้รับผิดชอบ)
- [ ] วันที่แผน vs วันที่จะไปจริง

### ขั้นตอนที่ 4: หา Cell Site ในพื้นที่

เปิด [ระบบ NSMC ของ กสทช.](https://nsmc.nbtc.go.th/) เพื่อค้นหาสถานีฐานโทรศัพท์มือถือในบริเวณที่จะไปวัด

**วิธีใช้:**

```
1. เปิด https://nsmc.nbtc.go.th/
2. ค้นหาตามจังหวัด/อำเภอ/ตำบล ที่จะไปวัด
3. ดูว่ามี Cell Site ของค่ายไหนบ้าง (AIS, TRUE, NT)
4. จดพิกัด + Site Code ของสถานีที่จะไปตรวจ
```

### ขั้นตอนที่ 5: เลือกจุดวัด EMF

!!! tip "หลักการเลือกจุดวัด"

    ไป **บริเวณที่มีคนอยู่อาศัย** ใกล้สถานีฐาน ไม่ใช่ไปวัดที่ตัวสถานี

**ลำดับการเลือก:**

```
1. หาพื้นที่ชุมชนก่อน:
   ├→ ตลาด
   ├→ โรงพยาบาล
   ├→ โรงเรียน
   ├→ วัด / ชุมชนหนาแน่น
   └→ สถานที่สาธารณะอื่นๆ

2. จากจุดนั้น หา Cell Site ที่ใกล้ที่สุด:
   └→ ใช้ NSMC หรือสังเกตเสาในพื้นที่

3. วัด EMF ตามมาตรฐาน 6 จุด (50-500 ม. จาก Cell Site)
   └→ ดูวิธีวัดละเอียดในหน้า [คู่มือ EMF](emf-guide.md)
```

### ขั้นตอนที่ 6: ตรวจป้าย Site Code ที่สถานีฐาน

เมื่อไปถึง Cell Site ต้องตรวจดูด้วยว่า **มีป้ายแสดงข้อมูลสถานี (Site Code)** หรือไม่

```
ตรวจป้ายของแต่ละค่าย:
  ├→ AIS    — มีป้าย Site Code ไหม?  ✅ / ❌
  ├→ TRUE   — มีป้าย Site Code ไหม?  ✅ / ❌
  └→ NT     — มีป้าย Site Code ไหม?  ✅ / ❌
```

!!! warning "ถ้าไม่มีป้าย Site Code"

    ถ้าพบว่าสถานีฐาน **ไม่มีป้ายแสดงข้อมูลสถานี** ตามที่กฎหมายกำหนด:

    1. ถ่ายรูปหลักฐาน (เสาที่ไม่มีป้าย)
    2. บันทึกพิกัด GPS
    3. จดรายละเอียด (ค่ายไหน, ที่ตั้ง, สภาพ)
    4. **ทำรายงานส่ง ภภ.2** เพื่อดำเนินการต่อ

### Checklist ก่อนออกจากจุดวัด EMF

- [ ] วัด EMF ครบ 6 จุด × 9 ตำแหน่ง
- [ ] บันทึกพิกัด GPS ทุกจุดวัด
- [ ] ถ่ายรูป Cell Site + เสา + ป้าย (หรือบันทึกว่าไม่มีป้าย)
- [ ] ถ่ายรูปบริเวณโดยรอบ (ตลาด, รพ, โรงเรียน ฯลฯ)
- [ ] ลงข้อมูลในระบบ OPER
- [ ] อัพเดท Status ใน Notion เป็น "Done"
- [ ] แจ้งทีมคุ้มครองสรุปผลเบื้องต้น

## 5. Work-Life Balance

!!! danger "สำคัญมาก!"

    **วางแผนให้ดี และ กลับให้ตรงเวลา**

    - ตั้งเวลากลับไว้ล่วงหน้า — อย่าให้งานกินเวลาส่วนตัว
    - ถ้าตรวจไม่ครบตามแผน ไม่เป็นไร — ย้ายไปวันถัดไปได้
    - พักผ่อนให้เพียงพอ โดยเฉพาะก่อนขับรถทางไกล
    - อย่าขับรถเร็ว — ปลอดภัยสำคัญกว่าตรวจครบ

### ตารางเวลาที่แนะนำ

| เวลา | กิจกรรม |
|------|---------|
| 07:30 | ออกจากสำนักงาน |
| 08:00 - 08:30 | ถึงสถานีแรก + เริ่มตรวจ |
| 08:30 - 11:30 | ตรวจสถานี 3-4 แห่ง |
| 11:30 - 13:00 | พักกลางวัน |
| 13:00 - 15:30 | ตรวจสถานีที่เหลือ 2-3 แห่ง |
| 15:30 - 16:00 | สรุปผล + ลงระบบ |
| 16:00 - 16:30 | เดินทางกลับสำนักงาน |
| **16:30** | **กลับถึง — ตรงเวลา!** |

!!! tip "จำไว้"

    งานตรวจสอบมีทุกเดือน ไม่ต้องรีบทำให้ครบในวันเดียว
    **สุขภาพกายและใจของทีม สำคัญกว่าตัวเลข KPI**
